## ðŸš€ How to Use

1.  **Open the App:** Click the launch link above.
2.  **Draw:** Drag shapes from the sidebar onto the canvas.
3.  **Save:** Click `File > Save` to download an editable `.svg` file to your computer.
4.  **Edit Later:** Drag that `.svg` file back onto the blank canvas to resume editing.

## ðŸ“¦ For Developers (Running Locally)

DashDraw is built on the robust open-source `draw.io` engine. If you want to run your own copy:

1.  Clone this repository:
    ```bash
    git clone [https://github.com/thevikramsinha/dashdraw.git](https://github.com/thevikramsinha/dashdraw.git)
    ```
2.  Navigate to the folder:
    ```bash
    cd dashdraw
    ```
3.  Open `index.html` in any modern web browser.
